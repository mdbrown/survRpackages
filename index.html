Prospective bio-marker validation with survival data

R packages

A suite of R packages for prospective bio-marker validation with time-to-event data developed in collaboration with Yingye Zheng. The packages are:

  • survAccuracyMeasures: Estimate the AUC, TPR(c), FPR(c), PPV(c), and NPV(c) for for a specific timepoint and marker cutoff value c. Standard errors and confidence intervals are also computed. Either analytic or bootstrap standard errors can be computed.

  • survCompetingRisk: Evaluate the prognostic accuracy of a marker with multiple competing risk events. Functions to calculate the AUC, ROC, PPV, and NPV are provided. A discrete covariate Z, if available, can be included.

  • survNRI: Calculate the net reclassification index (NRI) statistic for survival data using five different estimators.
